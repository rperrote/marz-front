# React Doctor ignores

`react-doctor` is run through `npx -y react-doctor@latest .` and there is no versioned project config file in this repo.

Current suppression strategy:

- Generated API code and TanStack Router output are excluded through `eslint.config.js` and `knip.jsonc`.
- Playwright fixtures in `src/test/e2e/**` use a fixture callback named `use`; ESLint now disables `react-hooks/rules-of-hooks` for that path so React Doctor consumes the same false-positive suppression.
